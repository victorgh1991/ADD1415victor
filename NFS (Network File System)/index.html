Comenzamos habilitando el rol o funcion de Servicios de Archivo






Una vez instalado, crearemos una carpeta y la compartiremos como NFS.






Ahora pasaremos a nuestro cliente, para que sea posible necesitaremos un Windows 7 Enterprise.



Habilitaremos la caracteristica de Servicios para NFS y dentro de él Cliente para NFS



Una vez ya habilitada la caracteristica, iniciamos el servicio en el cliente, y nos conectamos a nuestro servidor.

Linux

    Lo primero que haremos sera instalar NFS en nuestro Linux.




    Crearemos una carpeta export y dentro de ella public y private, con sus respectivos permisos de carpeta.



    Modificaremos el archivo exports, que se encuentra en /etc/, añadiremos al final del archivo las rutas de las carpetas, y los permisos que tendran los clientes que se conecten.



    Una vez completados estos pasos, iniciaremos el servicio de NFS



    Comprobaremos que se han añadido bien las carpetas que exportaremos con este comando



    Ahora pasamos a nuestro cliente, le instalamos primero el paquete de NFS.



    Comprobaremos que nuestro cliente y servidor con capaces de verse usando un ping para demostrarlo.



    Con un nmap mas la ip de nuestro servidor comprobamos que el puerto de NFS esta abierto correctamente.



    Crearemos las carpetas en el cliente y montaremos las carpetas del servidor en el cliente.



    Y comprobamos que dentro de las carpetas estan los datos que creamos en el servidor, dentro de las carpetas del cliente.




    Como ultimo paso haremos que nuestro cliente monte las carpetas automaticamente modificando el archivo fstab que se encuentra en /etc/.




    Una vez configurado fstab, reiniciamos el sistema, y comprobamos que las carpetas han sido montadas con un df -hT.
